Encryption Key
Code

This program started as a challenge given to me by a friend. The goal was to use Java to make a simple encryption and decription software. Using a key, the user can encrypt or decrypt a string. The program can also generate an encryption key for the user.

Black EP
Music

The Black EP is an unfinished EP that explored mixing Lo-Fi music with different genres like Rock & Roll or Trap for example. Only two tracks were made for it, one exploring Lo-Fi and Electronic music, and the other one exploring Lo-Fi and Bedroom Pop.

Demo EP
Music

The Demo EP is an unreleased EP made for my application process for the Université de Montréal's Bachelor in music production. It was made of 4 tracks which all showed a different skill related to music production: composing melody, creating sounds, writing drums and fliping samples.
